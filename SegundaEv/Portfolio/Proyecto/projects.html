PROYECTOS FLORIDA


Aquí hay un recopilatorio de los proyectos que he llevado para el curso de 2º de DAW (2023-2024). El repositorio está dividió en ramas para cada asignatura, con proyectos de todo tipo: FrontEnd, BackEnd, Diseño, etc. Permite ver una evolución de mi forma de programar a lo largo del año escolar.

InCube


InCube es un proyecto integrado que realice junto con un grupo de compañeros de clase a final de DAM. El proyecto se trata de una aplicación móvil que envía señales a una placa Arduino que pone a funcionar la maqueta de una incubadora. En este proyecto me encargué de desarrollar el backend y la api junto con un compañero, así como administrar la base de datos no-sql (MongoDB).

Memigo


Memigo es el proyecto final que lleve a cabo durante el último trimestre del curso de DAM, se trata de una aplicación basada en instagram y memedroid que sirve para ver y crear memes. La aplicación permite ver los memes que se suben a la base de datos, utilizando una api en C# como intermediario. Utiliza una base de datos MariaDB y un FrontEnd hecho en ReactNative.
