Crear el contenedor con el composer: docker run --rm --interactive --tty --volume .:/app composer init
Añadir Twig: docker run --rm --interactive --tty --volume .:/app composer require twig
Añadir Doctrine: docker run --rm --interactive --tty --volume .:/app composer require symfony/orm-pack
En el .env creado en symfony sustituimos la linea 29 (comentarla) por la linea de DATABASE_URL proporcionada por Jorge.
